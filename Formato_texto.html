Empezando con la Web

Introducción a la Webes una serie concisa que le presenta los aspectos prácticos del desarrollo web. Configurará las herramientas que necesita para construir una página web simple y publicar su propio código simple.


La historia de tu primer sitio web

Es mucho trabajo crear un sitio web profesional, así que si eres nuevo en el desarrollo web, te animamos a que empieces poco a poco. No creará otro Facebook de inmediato, pero no es difícil crear su propio sitio web sencillo en línea, así que comenzaremos por ahí.

Al trabajar con los artículos que se enumeran a continuación, pasará de la nada a tener su primera página web en línea. ¡Comencemos nuestro viaje!

Instalación de software básico

Cuando se trata de herramientas para crear un sitio web, hay muchas para elegir. Si recién está comenzando, es posible que se sienta confundido por la variedad de editores de código, marcos y herramientas de prueba que existen. En Instalación de software básico Instalación de software básico , le mostraremos paso a paso cómo instalar el software que necesita para comenzar un desarrollo web básico.


¿Cómo se verá su sitio web?

Antes de comenzar a escribir el código para su sitio web, primero debe planificarlo. ¿Qué información estás mostrando? ¿Qué fuentes y colores estás usando? ¿Cómo se verá su sitio web? Describimos un método simple que puede seguir para planificar el contenido y el diseño de su sitio.


Tratar con archivos

Un sitio web consta de muchos archivos: contenido de texto, código, hojas de estilo, contenido multimedia, etc. Cuando está creando un sitio web, necesita ensamblar estos archivos en una estructura sensata y asegurarse de que puedan comunicarse entre sí. El manejo de archivos explica cómo configurar una estructura de archivos sensible para su sitio web y qué problemas debe tener en cuenta.


Conceptos básicos de HTML

El lenguaje de marcado de hipertexto (HTML) es el código que utiliza para estructurar su contenido web y darle significado y propósito. Por ejemplo, ¿mi contenido es un conjunto de párrafos o una lista de viñetas? ¿Tengo imágenes insertadas en mi página? ¿Tengo una tabla de datos? Sin abrumarlo, los conceptos básicos de HTML brindan suficiente información para familiarizarlo con HTML.


Conceptos básicos de CSS

Hojas de estilo en cascada (CSS) es el código que utiliza para diseñar su sitio web. Por ejemplo, ¿desea que el texto sea negro o rojo? ¿Dónde se debe dibujar el contenido en la pantalla? ¿Qué imágenes de fondo y colores se deben utilizar para decorar su sitio web? Los conceptos básicos de CSS lo guiarán a través de lo que necesita para comenzar.


Conceptos básicos de JavaScript

JavaScript es el lenguaje de programación que utiliza para agregar funciones interactivas a su sitio web. Algunos ejemplos podrían ser juegos, cosas que suceden cuando se presionan botones o se ingresan datos en formularios, efectos de estilo dinámico, animación y mucho más. Los conceptos básicos de JavaScript le dan una idea de lo que es posible con este interesante lenguaje y cómo empezar.


Publicando su sitio web

Una vez que haya terminado de escribir el código y organizar los archivos que componen su sitio web, debe ponerlo todo en línea para que la gente pueda encontrarlo. La publicación de su código de muestra describe cómo obtener su código de muestra simple en línea con el mínimo esfuerzo.


Como funciona la web

Cuando accedes a tu sitio web favorito, suceden muchas cosas complicadas en segundo plano que es posible que no conozcas. El funcionamiento de la web describe lo que sucede cuando visualiza una página web en su computadora.
